# Sistema de Gestão de Banco de Dados - Lan House 🎮

Projeto acadêmico desenvolvido na disciplina de Programação e Otimização de Banco de Dados, focado na modelagem, normalização e implementação de um sistema relacional completo para uma Lan House.

## 🗂️ Estrutura do Repositório

* **`SCHEMA.SQL`**: Scripts DDL para criação do banco de dados `LanHouse_DB`, estruturação das tabelas e definição de constraints (PK, FK, UNIQUE).
* **`DATA.SQL`**: Scripts DML contendo as inserções de dados iniciais (carga inicial) para testes do sistema.
* **`RELATORIOS.SQL`**: Scripts DQL contendo queries avançadas utilizando `JOIN`, `GROUP BY`, `HAVING` e funções de agregação para geração de relatórios estratégicos.
* **`MODELO_CONCEITUAL.pdf/png`**: Diagrama Entidade-Relacionamento (DER) mapeando as entidades e cardinalidades.
* **`MODELO_LOGICO.pdf/png`**: Diagrama relacional detalhando chaves primárias, estrangeiras e tipos de dados.

## 🛠️ Tecnologias Utilizadas
* MySQL
* Diagramação de Modelos de Dados (MER/DER)
